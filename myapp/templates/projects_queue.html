Simulated Networks and obtained Performance Metrics.

Overview:

This Project was done as a part of Performance of Computer Networks course at UTD during Fall 2014.

Languages and Tools:

  • C++
  • Netbeans IDE

Implementation:

  • Event Driven Simulation of an Open Queuing Network. An open queuing network was simulated wherein packets/customers could come from outside the network or served again. By the end of the simulation, various performance metrics like throughput, expected waiting time etc. in the system are calculated. By analyzing the metrics, a general understanding of how packets/customers are being served in the system is known.
  • Event Driven Simulation of a Web Server modeled as an M|M|2|4 Queuing System. The performance metrics for a finite capacity multiple server system,at various conditions, were obtained. These metrics were later used to analyse the network performance so that the reasons for that particular performance is known.

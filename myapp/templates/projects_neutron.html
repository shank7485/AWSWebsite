Rolling upgrades in OpenStack Neutron.

Neutron Docs: https://docs.openstack.org/neutron/latest/contributor/internals/upgrade.html

Overview:

This was done when I was an upstream contributor to OpenStack Neutron as a part of the Neutron upgrades team.

The effort in the community was mainly aimed at enabling rolling upgrades in OpenStack Neutron by making using of Oslo Versioned Objects (OVOs) to achieve versioning on RPCs when there are mixed (n and n+1) clients and servers in an OpenStack cloud system when undergoing upgrades.

I was personally responsible for implementing and integrating various OVOs into core Neutron codebase.

Tech stack:

  • OpenStack Neutron
  • Python
  • Oslo Versioned Objects from OpenStack Oslo Library

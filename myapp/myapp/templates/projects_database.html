Designed a Database for a Flea Market.

Overview:

This Project was done as a part of Database Design course at UTD during Spring 2015.

The project was mainly aimed to design and implement a database to be used in a Flea Market to maintain the stall spaces.

Languages and Tools:

  • Oracle DB
  • SQL

Working:

  • This database is mainly used by the Managers of a Flea Market to maintain the stallâ€™s rent, information of renter, objects sold in the stalls, maintenance staff etc.
  • Analyzed the Data requirements.
  • Formed ER Diagrams and mapped it into Relations (Tables) and assigned various Primary, Foreign keys.
  • Normalized the Relations to 3NF form.
  • Created the relations on Oracle DB using SQL CREATE statements.
  • Populated the Relations with example records using SQL INSERT statements.
  • Various updates to this database can be done using SQL UPDATE statements.
